# vue3docs-guides

A compilation of vue guides compiled in an SPA.
CSS library used: TailwindCSS

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
